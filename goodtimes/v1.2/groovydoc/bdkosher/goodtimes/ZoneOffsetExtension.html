        • static java.util.Map<java.time.temporal.TemporalUnit, java.lang.Integer> describe(java.time.ZoneOffset self)

          Describes this ZoneOffset as a Map of ChronoUnit keys (HOURS, MINUTES, and SECONDS) and integer values. The values are normalized. For example, ZoneOffset.ofTotalSeconds(60 * 60 * 3 + 60 * 15).describe() would return [ChronoUnit.HOURS) : 3, (ChronoUnit.MINUTES) : 15, (ChronoUnit.SECONDS) : 0]. If a ZoneOffset is negative, then all of the non-zero fields will be negative.
