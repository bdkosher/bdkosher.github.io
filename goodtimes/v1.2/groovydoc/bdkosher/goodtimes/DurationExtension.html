        • static java.util.Map<java.time.temporal.TemporalUnit, java.lang.Long> describe(java.time.Duration self)

          Describes this Duration as a Map of ChronoUnit keys (HOURS, MINUTES, SECONDS, and NANOS) and long values. The values are normalized. For example, Duration.ofSeconds(62).describe() would return [ChronoUnit.HOURS) : 0, (ChronoUnit.MINUTES) : 1, (ChronoUnit.SECONDS) : 2, ChronoUnit.NANOS) : 0]. If a Duration is negative, then all of the non-zero fields will be negative.
